# QUEUESTOCK SYSTEM

A comprehensive inventory and order management system designed to streamline stock management and customer orders.

## Tech Stack

### Frontend
- **React** - UI library
- **Vite** - Build tool and dev server
- **CSS** - Styling

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework

## Features

- **Admin Dashboard**
  - Add and manage products
  - Manage stock levels
  - View and process customer orders
  - User management

- **Customer Dashboard**
  - Browse available products
  - Place orders
  - View order history
